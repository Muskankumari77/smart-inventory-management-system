# Smart Inventory Management System

A Flipkart-style inventory management system built using Python Flask.

## Features
- Add new products with quantity and threshold
- Dashboard showing total, low stock, and healthy stock
- Low-stock alert banner
- Search products by name
- Delete products with confirmation
- CSV-based data storage

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- CSV
- Git & GitHub

## How to Run
1. Clone the repository
2. Install Flask using `pip install flask`
3. Run the app using `python app.py`
4. Open browser and go to `http://127.0.0.1:5000/`

## Use Case
This project simulates a real-world inventory system used in e-commerce platforms like Flipkart.
